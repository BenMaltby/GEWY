# GEWY
Basic, light-weight Imperative GUI library for pygame

Requires included file: VOBJ.py

<img src="GEWY_demo_image.png" width="600" height="525">

# Features:
 - Buttons
 - Variable Sliders
 - Colour Sliders
 - Pie Charts
 - Tab System
 - Moveable windows
